# xim-pkgindex-scode

xim package index for source code

## 添加索引仓库

> 添加索引仓库到xim的索引仓库管理器中

```bash
xim --add-indexrepo scode:https://github.com/d2learn/xim-pkgindex-scode.git
```

## 使用仓库中的包

```bash
xlings install scode:linux
```
