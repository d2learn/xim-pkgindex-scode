# xim-pkgindex-template

xim package-index - template repository

## 添加索引仓库

> 添加索引仓库到xim的索引仓库管理器中

```bash
xim --add-indexrepo namespace:https://github.com/d2learn/xim-pkgindex-template.git
```

## 使用仓库中的包

```bash
xlings install namespace:pkgname
```
或
```bash
xim -i namespace:pkgname
```
